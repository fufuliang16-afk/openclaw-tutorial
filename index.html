安装指南

快速安装

# 使用npm安装
npm install -g openclaw

# 或者使用npx
npx openclaw@latest configure

详细步骤

  1. 确保已安装Node.js (v18+)
  2. 安装OpenClaw全局包
  3. 运行配置向导
  4. 选择需要的插件和功能
  5. 开始使用！
