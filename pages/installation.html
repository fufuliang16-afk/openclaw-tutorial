OpenClaw安装指南

本指南将帮助你完成OpenClaw的安装和基本配置。

系统要求

基本要求
  • Node.js v18.0.0 或更高版本
  • npm v9.0.0 或更高版本
  • 操作系统: macOS, Linux, Windows
  • 内存: 至少 2GB RAM
  • 磁盘空间: 至少 500MB 可用空间
推荐配置
  • Node.js v20+（LTS版本）
  • 终端: iTerm2, Windows Terminal
  • Shell: zsh, bash, PowerShell
  • 网络: 稳定的互联网连接
  • 编辑器: VS Code, Sublime Text

快速安装

如果你已经满足系统要求，可以使用以下命令快速安装：

安装命令 
# 使用npm全局安装
npm install -g openclaw

# 或者使用npx（无需安装）
npx openclaw@latest configure

# 验证安装
openclaw --version
提示：使用 npx 可以无需全局安装直接运行最新版本的OpenClaw。

详细安装步骤

首先检查你的Node.js版本：

node --version
# 应该显示 v18.0.0 或更高版本

npm --version
# 应该显示 v9.0.0 或更高版本

如果版本过低，请访问 Node.js官网 下载最新LTS版本。

使用npm全局安装OpenClaw：

# 全局安装（推荐）
npm install -g openclaw

# 或者安装特定版本
npm install -g openclaw@latest

如果遇到权限问题，可以尝试：

# 使用sudo（macOS/Linux）
sudo npm install -g openclaw

# 或者修改npm全局目录权限
npm config set prefix ~/.npm-global
echo 'export PATH=~/.npm-global/bin:$PATH' >> ~/.bashrc
source ~/.bashrc
npm install -g openclaw

安装完成后，运行配置向导：

# 运行配置向导
openclaw configure

# 或者使用交互式向导
openclaw onboard

配置向导会引导你完成：

  1. 选择工作空间目录
  2. 配置AI模型提供商
  3. 设置通信渠道
  4. 安装插件
  5. 创建初始配置文件

安装并配置完成后，启动OpenClaw网关：

# 启动网关服务
openclaw gateway start

# 检查网关状态
openclaw gateway status

# 查看日志
openclaw logs --follow

网关启动后，OpenClaw就可以开始工作了！

平台特定指南

macOS
  • 使用Homebrew安装Node.js：
    brew install node
  • 推荐使用iTerm2作为终端
  • 可能需要配置PATH：
    echo 'export PATH="/usr/local/bin:$PATH"' >> ~/.zshrc
Linux
  • 使用包管理器安装Node.js：
    Ubuntu: sudo apt install nodejs npm
    Fedora: sudo dnf install nodejs npm
  • 可能需要安装build-essential：
    sudo apt install build-essential
Windows
  • 从Node.js官网下载安装包
  • 使用Windows Terminal或PowerShell
  • 可能需要以管理员身份运行终端
  • 考虑使用WSL2获得更好的体验

常见问题与故障排除

原因：npm全局目录不在PATH中

解决方案：

# 查找npm全局目录
npm config get prefix

# 将目录添加到PATH（macOS/Linux）
echo 'export PATH="$(npm config get prefix)/bin:$PATH"' >> ~/.zshrc
source ~/.zshrc

# Windows: 将npm全局目录添加到系统PATH环境变量

原因：没有权限写入npm全局目录

解决方案：

# 方法1: 使用sudo（不推荐）
sudo npm install -g openclaw

# 方法2: 修改npm全局目录所有权（macOS/Linux）
sudo chown -R $(whoami) $(npm config get prefix)/{lib/node_modules,bin,share}

# 方法3: 使用nvm管理Node.js版本
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
nvm install --lts
nvm use --lts
npm install -g openclaw
