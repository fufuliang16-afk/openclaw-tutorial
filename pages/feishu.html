OpenClaw飞书集成教程

基于实际经验，手把手教你配置OpenClaw与飞书的集成。

概述

为什么集成飞书？
  • 企业级通信 - 在企业环境中使用
  • 移动办公 - 通过手机随时访问
  • 团队协作 - 在群聊中使用AI助手
  • 自动化工作流 - 集成企业应用
集成功能
  • 实时聊天 - 在飞书中与OpenClaw对话
  • 文件操作 - 访问飞书文档和云盘
  • 通知推送 - 接收任务通知
  • 群组管理 - 在团队群聊中提供帮助
本教程基于实际配置经验编写，所有步骤都经过实际测试验证。

准备工作

所需条件
软件要求
  • ✅ OpenClaw 已安装并运行
  • ✅ Node.js v18+
  • ✅ 稳定的网络连接
  • ✅ 飞书账号（企业版）
账号要求
  • ✅ 飞书开放平台账号
  • ✅ 企业管理员权限（部分功能）
  • ✅ 可创建企业自建应用
提示：如果你还没有飞书企业账号，可以访问 飞书官网 注册试用。

创建飞书应用

访问飞书开放平台：

https://open.feishu.cn/app

使用你的飞书账号登录。

注意：国际版用户请访问 https://open.larksuite.com/app

  1. 点击"创建企业自建应用"
  2. 填写应用信息：
    • 应用名称：OpenClaw助手
    • 应用描述：AI个人助手
    • 应用图标：上传或选择图标
  3. 点击"确定创建"

飞书创建应用界面示意图

创建应用后，进入"凭证与基础信息"页面：

重要信息：
  • App ID：格式如 cli_xxx
  • App Secret：长字符串，妥善保管
示例：
App ID: cli_a92d073398385bc2
App Secret: pNIXFyff9GVRInlG3PDMpb8UKeEEAkGT
安全警告：App Secret 相当于密码，不要泄露给他人！

在"权限管理"页面，点击"批量导入"，粘贴以下权限配置：

{
  "scopes": {
    "tenant": [
      "aily:file:read",
      "aily:file:write",
      "application:application.app_message_stats.overview:readonly",
      "application:application:self_manage",
      "application:bot.menu:write",
      "cardkit:card:read",
      "cardkit:card:write",
      "contact:user.employee_id:readonly",
      "corehr:file:download",
      "event:ip_list",
      "im:chat.access_event.bot_p2p_chat:read",
      "im:chat.members:bot_access",
      "im:message",
      "im:message.group_at_msg:readonly",
      "im:message.p2p_msg:readonly",
      "im:message:readonly",
      "im:message:send_as_bot",
      "im:resource"
    ],
    "user": ["aily:file:read", "aily:file:write", "im:chat.access_event.bot_p2p_chat:read"]
  }
}
这些权限包含了聊天、文件操作等基本功能。根据你的需求可以增减。

配置OpenClaw

方法1：使用命令行配置 
# 运行配置向导
openclaw channels add

# 选择 Feishu 渠道
# 输入你的 App ID 和 App Secret
# 完成配置
方法2：手动编辑配置文件

编辑OpenClaw配置文件：

# 配置文件位置
~/.openclaw/openclaw.json

添加飞书配置：

{
  "channels": {
    "feishu": {
      "enabled": true,
      "appId": "你的_App_ID",
      "appSecret": "你的_App_Secret",
      "domain": "feishu",
      "groupPolicy": "open",
      "dmPolicy": "pairing",
      "accounts": {
        "main": {
          "appId": "你的_App_ID",
          "appSecret": "你的_App_Secret",
          "botName": "OpenClaw助手",
          "domain": "feishu"
        }
      },
      "typingIndicator": true,
      "resolveSenderNames": true
    }
  }
}
注意：编辑配置文件后需要重启OpenClaw网关： openclaw gateway restart

配置事件订阅（关键步骤）

这是最关键的步骤！在飞书开放平台配置事件订阅：

  1. 进入"事件与回调"页面
  2. 在"订阅方式"选择"使用长连接接收事件/回调"
  3. 添加事件：im.message.receive_v1
  4. 保存配置
重要：必须选择"长连接"模式！Webhook模式需要公网IP，不适合大多数用户。
为什么需要事件订阅？
  • 让OpenClaw能够接收飞书消息</
